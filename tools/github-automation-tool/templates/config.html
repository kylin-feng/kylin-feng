配置说明
📁 仓库路径

输入你要自动化管理的GitHub仓库在本地的完整路径。确保该路径存在且已初始化为Git仓库。

🌐 远程仓库

通常为"origin"，如果你有多个远程仓库，可以指定其他名称。

🔀 分支名称

要提交的目标分支，通常为"main"或"master"。

☁️ 自动推送

启用后会自动将提交推送到GitHub，关闭则只在本地提交。
