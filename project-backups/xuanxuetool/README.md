# 玄学算命网站

一个现代化的玄学算命网站，提供塔罗牌占卜、八字算命和星座运势等功能。

## 功能特色

### 🔮 塔罗牌占卜
- 12张精选塔罗牌
- 详细的牌意解释
- 个性化的建议和指导
- 精美的卡片动画效果

### 📅 八字算命
- 基于出生日期和时间的八字计算
- 天干地支分析
- 性格特征分析
- 事业、感情、健康建议

### ⭐ 星座运势
- 12星座完整运势
- 综合运势、爱情、事业、财运、健康分析
- 每日建议和指导
- 精美的星座图标

## 技术特点

- **现代化设计**: 采用渐变背景和毛玻璃效果
- **响应式布局**: 完美适配桌面端和移动端
- **流畅动画**: CSS3动画和过渡效果
- **交互体验**: 丰富的用户交互反馈
- **纯前端**: 无需服务器，直接在浏览器中运行

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 选择你感兴趣的算命方式：
   - 点击"塔罗牌"标签进入塔罗牌占卜
   - 点击"八字算命"标签进行八字分析
   - 点击"星座运势"标签查看星座运势

### 塔罗牌占卜
1. 点击任意一张塔罗牌
2. 查看详细的牌意解释
3. 阅读个性化建议

### 八字算命
1. 输入你的出生日期和时间
2. 选择性别
3. 点击"开始分析"查看八字结果

### 星座运势
1. 选择你的星座
2. 查看详细的运势分析
3. 阅读每日建议
