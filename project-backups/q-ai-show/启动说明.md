# 小程序开发服务展示平台

## 🚀 快速启动

### 1. 安装依赖
```bash
npm install
# 或
yarn install
```

### 2. 启动开发服务器
```bash
# 微信小程序开发模式
npm run dev:weapp

# H5开发模式  
npm run dev:h5
```

### 3. 构建发布
```bash
# 构建微信小程序
npm run build:weapp

# 构建H5版本
npm run build:h5
```

## 🎨 设计特色

- **现代化UI**: 采用渐变背景和毛玻璃效果
- **响应式设计**: 适配各种屏幕尺寸
- **专业配色**: 蓝紫色渐变主题
- **交互友好**: 流畅的动画和过渡效果

## 📦 技术栈

- **框架**: Taro 3.6+
- **语言**: TypeScript
- **样式**: Sass/SCSS
- **UI组件**: Taro Components
- **构建工具**: Webpack 5

## 🔧 自定义配置

### 修改个人信息
编辑 `src/pages/home/index.tsx` 中的个人信息

### 添加项目案例
编辑 `src/pages/showcase/index.tsx` 中的项目数据

### 修改服务套餐
编辑 `src/pages/services/index.tsx` 中的服务数据

### 更新联系方式
编辑 `src/pages/contact/index.tsx` 中的联系信息
