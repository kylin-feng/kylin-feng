# macOS 内置OCR快捷指令设置

## 方法一：使用快捷指令App (推荐)

### 步骤：
1. 打开"快捷指令"App (Shortcuts)
2. 点击右上角的"+"新建快捷指令
3. 搜索并添加以下操作：
   - "截屏" (Take Screenshot)
   - "从图像中获取文本" (Get Text from Image)
   - "拷贝到剪贴板" (Copy to Clipboard)

### 设置快捷键：
1. 快捷指令完成后，点击设置图标
2. 选择"添加到Dock"或"钉住快捷指令"
3. 在"系统偏好设置" > "键盘" > "快捷键" > "App快捷键"
4. 添加快捷指令App，设置快捷键如 `Cmd+Shift+T`

## 方法二：使用Automator

### 步骤：
1. 打开Automator
2. 新建"快速操作"
3. 添加操作：
   - "截取屏幕区域"
   - "提取图像文本"
   - "拷贝到剪贴板"
4. 保存为"OCR截图"

### 设置快捷键：
在"系统偏好设置" > "键盍" > "快捷键" > "服务"中找到你创建的服务，设置快捷键

## 快速体验：
- 按 `Cmd+Shift+4` 截图
- 在预览中打开图片
- 按 `Cmd+A` 全选
- 按 `Cmd+C` 复制（会自动OCR识别文字）
