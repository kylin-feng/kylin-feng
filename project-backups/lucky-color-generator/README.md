# 幸运色生成器

一个现代化的幸运色生成器Web应用，包含主页和颜色生成两个页面。

## 功能特色

- 🎨 **智能生成**: 基于算法智能生成每日幸运色
- 🌈 **丰富色彩**: 12种精选颜色，每种都有独特的寓意和能量
- 📱 **便捷使用**: 简洁直观的界面，随时随地获取幸运色
- ✨ **每日更新**: 每日不同的幸运色组合，保持新鲜感
- 📊 **历史记录**: 保存生成历史，方便查看
- 📤 **分享功能**: 支持分享幸运色结果

## 使用方法

### 1. 启动后端服务

```bash
cd lucky
python backend.py
```

后端服务将在 `http://10.10.100.19:5001` 启动

### 2. 访问前端页面

- 主页: `http://localhost:8080/index.html`
- 颜色生成: `http://localhost:8080/color-generator.html`

### 3. 生成幸运色

1. 点击"生成幸运色"按钮
2. 等待API响应或本地生成
3. 查看生成的幸运色和寓意
4. 可选择分享或重新生成

## API接口

### 获取幸运色
- **URL**: `/api/lucky-color`
- **方法**: GET
- **响应**: 
```json
{
  "color": [1, 3],
  "description": "今日幸运色组合：绿色 + 红色，多重好运加持，事事顺心如意"
}
```

### 获取颜色定义
- **URL**: `/api/colors`
- **方法**: GET
- **响应**: 包含所有颜色定义和描述的JSON

### 健康检查
- **URL**: `/health`
- **方法**: GET
- **响应**: `{"status": "healthy"}`
