# 鸿蒙项目构建指南

## 1. 在DevEco Studio中构建
1. 打开DevEco Studio (已运行)
2. 项目已加载在: /Users/shixianping/2025-10/app-hm
3. 在菜单栏选择: Build -> Build Hap(s)/App(s) 
4. 或使用快捷键构建

## 2. 构建产物位置
构建完成后，HAP文件将在以下位置:
- build/default/outputs/default/entry-default-signed.hap

## 3. 如果遇到问题
- 确保SDK路径正确: /Users/shixianping/Library/Huawei/Sdk
- 检查签名配置
- 清理并重新构建
