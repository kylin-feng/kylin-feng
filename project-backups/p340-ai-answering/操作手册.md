# ultraArm P340 机械臂操作手册

## 一、硬件连接

1. 机械臂通电（接电源适配器）
2. USB 数据线连接机械臂与 MacBook
3. 确认串口出现：
   ```bash
   ls /dev/cu.*
   # 应看到 /dev/cu.wchusbserial1140
   ```

## 五、控制台操作说明

### 键盘模式（默认）
| 按键 | 动作 |
|------|------|
| `← →` 或 `A D` | J1 旋转（左右） |
| `↑ ↓` 或 `W S` | J2 仰角（上下） |
| `Q E` | J3 伸展 / 收回 |
| `Z X` | J4 腕转 |
| `Space` | 回零位 |
| `G` | 切换为手势模式 |
| `K` | 切换为键盘模式 |

### 手势模式
| 动作 | 控制 |
|------|------|
| 食指尖上下移动 | J1 旋转 |
| 食指尖左右移动 | J2 仰角 |
| 拇指食指捏合/张开 | J3 伸展 |
| 手腕顺/逆时针旋转 | J4 腕转 |
| 手移出画面 | 机械臂保持原位 |

## 七、常见问题

### 串口找不到 / 通信无响应
1. 确认机械臂已通电
2. 重新插拔 USB 线
3. 执行 `ls /dev/cu.*` 确认串口名称
4. 更新 `config.yaml` 中的 `com_port`

### 摄像头无画面
- 检查「系统设置 → 隐私与安全性 → 摄像头」，给终端授权

### 端口 5001 被占用
```bash
lsof -i :5001
kill -9 <PID>
```

### 机械臂不动但通信正常
- 确认机械臂已通电（仅 USB 不够，必须同时接电源）
- macOS 需安装 CH340 驱动才能正常通信

## 八、快速测试脚本

```python
from pymycobot.ultraArmP340 import ultraArmP340
import time

ua = ultraArmP340('/dev/cu.wchusbserial1140', 115200)
ua.go_zero()
ua.set_angles([0, 0, 0], 50)
time.sleep(2.5)
# 左转
ua.set_angle(1, 45, 50)
time.sleep(2)
# 回零
ua.set_angles([0, 0, 0], 50)
```
