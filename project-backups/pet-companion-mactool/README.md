# 🐾 PetCompanion - macOS桌面宠物

一个可爱的macOS桌面宠物应用，可以说话、动画和互动！

## ✨ 功能特点

- 🐱 **多种宠物类型**：猫咪、小狗、兔子、小鸟
- 🎭 **丰富的心情系统**：开心、难过、兴奋、困倦、生气
- 🗣️ **语音合成**：宠物可以说话，支持多种语音和语速调节
- 🎨 **精美动画**：流畅的动画效果和交互反馈
- ⚙️ **个性化设置**：可自定义宠物名称、类型、语音参数等

## 🚀 快速开始

### 系统要求
- macOS 14.0 或更高版本
- Xcode 15.0 或更高版本

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <repository-url>
   cd PetCompanion
   ```

2. **打开项目**
   ```bash
   open PetCompanion.xcodeproj
   ```

3. **编译运行**
   - 在Xcode中选择目标设备
   - 按 `Cmd + R` 运行项目

## 🎮 使用指南

### 基本操作
- **播放/暂停**：点击播放按钮控制宠物动画
- **说话**：点击说话按钮让宠物随机说话
- **心情**：点击心情按钮切换宠物心情状态
- **设置**：点击齿轮图标打开设置界面

### 设置选项
- **宠物设置**：修改宠物名称和类型
- **语音设置**：调整语速、音调和语音类型
- **心情设置**：手动选择宠物心情
- **测试语音**：测试当前语音设置

## 🛠️ 技术实现

### 架构设计
- **SwiftUI**：现代化的用户界面框架
- **MVVM模式**：清晰的代码结构和数据绑定
- **AVFoundation**：语音合成和音频处理
- **Combine**：响应式编程和数据流管理

### 核心组件
- `PetModel`：宠物数据模型和状态管理
- `SpeechManager`：语音合成管理器
- `PetView`：宠物显示和动画组件
- `SettingsView`：设置界面

## 🎨 自定义开发

### 添加新的宠物类型
1. 在 `PetModel.swift` 中的 `PetType` 枚举添加新类型
2. 添加对应的emoji和显示名称
3. 更新UI显示逻辑

### 添加新的心情状态
1. 在 `PetModel.swift` 中的 `PetMood` 枚举添加新心情
2. 设置对应的emoji、颜色和消息
3. 更新动画和交互逻辑

### 扩展语音功能
1. 在 `SpeechManager.swift` 中添加新的语音选项
2. 实现语音识别功能
3. 添加语音命令处理

## 🔧 故障排除

### 常见问题
1. **语音不工作**：检查系统语音设置和权限
2. **动画卡顿**：降低动画复杂度或检查系统性能
3. **设置不保存**：确保应用有写入权限
