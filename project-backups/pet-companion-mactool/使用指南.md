# 🐾 PetCompanion 使用指南

## 📱 应用介绍

PetCompanion是一个可爱的macOS桌面宠物应用，让你的桌面更加生动有趣！这个应用包含了一个可以说话、动画和互动的小宠物。

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）
```bash
# 在项目目录中运行
./run.sh
```

### 方法二：手动运行
1. 双击 `PetCompanion.xcodeproj` 文件
2. 在Xcode中按 `Cmd + R` 运行项目

## 🎮 功能详解

### 主界面功能

#### 🎬 播放/暂停按钮
- **功能**：控制宠物的动画效果
- **效果**：宠物会进行弹跳、旋转和缩放动画
- **使用**：点击播放按钮开始动画，再次点击暂停

#### 🗣️ 说话按钮
- **功能**：让宠物随机说话
- **内容**：宠物会说出预设的可爱话语
- **语音**：使用系统语音合成技术

#### ❤️ 心情按钮
- **功能**：切换宠物的心情状态
- **心情类型**：
  - 😊 开心 - 黄色主题
  - 😢 难过 - 蓝色主题
  - 🤩 兴奋 - 橙色主题
  - 😴 困倦 - 紫色主题
  - 😠 生气 - 红色主题

#### ⚙️ 设置按钮
- **功能**：打开详细设置界面
- **位置**：右上角齿轮图标

### 设置界面详解

#### 🐾 宠物设置
- **宠物名称**：自定义你的宠物名字
- **宠物类型**：选择宠物种类
  - 🐱 猫咪
  - 🐶 小狗
  - 🐰 兔子
  - 🐦 小鸟

#### 🎵 语音设置
- **语速调节**：控制宠物说话的语速（10%-100%）
- **音调调节**：调整语音的音调高低（50%-200%）
- **语音选择**：选择不同的语音类型
  - TingTing（中文女声）
  - Sin-ji（中文女声）
  - Yu-shu（中文女声）
  - Alex（英文男声）
  - Samantha（英文女声）

#### 🎭 心情设置
- **手动选择**：点击心情图标直接切换
- **实时预览**：选择后立即生效
- **颜色主题**：每种心情都有对应的颜色主题

#### 🧪 测试语音
- **测试当前设置**：使用当前配置测试语音
- **停止播放**：停止当前语音播放
- **实时调整**：修改设置后可以立即测试效果

## 🔧 技术特性

### 系统要求
- **操作系统**：macOS 14.0 或更高版本
- **开发工具**：Xcode 15.0 或更高版本
- **内存要求**：至少4GB RAM
- **存储空间**：约50MB

### 权限要求
- **语音合成**：需要系统语音权限
- **音频播放**：需要音频输出权限
- **文件访问**：需要设置文件读写权限

## 🛠️ 故障排除

### 常见问题

#### 1. 语音不工作
**症状**：点击说话按钮没有声音
**解决方案**：
- 检查系统音量设置
- 确认语音合成权限已授予
- 尝试更换语音类型
- 重启应用

#### 2. 动画卡顿
**症状**：宠物动画不流畅
**解决方案**：
- 关闭其他占用GPU的应用
- 降低系统动画效果
- 重启应用

#### 3. 设置不保存
**症状**：重启后设置恢复默认
**解决方案**：
- 检查应用权限设置
- 确保有文件写入权限
- 重新设置并保存

#### 4. 编译错误
**症状**：Xcode编译失败
**解决方案**：
- 检查Xcode版本（需要15.0+）
- 检查macOS版本（需要14.0+）
- 清理项目缓存（Product → Clean Build Folder）
- 重新打开项目

### 调试技巧
- 使用Xcode的调试工具查看错误信息
- 检查控制台输出
- 验证系统权限设置
- 查看系统日志
