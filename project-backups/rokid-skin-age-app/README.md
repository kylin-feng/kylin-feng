# 皮肤年龄估算应用

一个简洁的Flutter应用，用于通过摄像头拍摄皮肤照片并估算年龄。

## 功能特点

- 横屏摄像头界面
- 简洁的拍照引导框
- AI皮肤年龄估算（模拟）
- 结果展示页面
- 支持重新拍照

## 技术栈

- Flutter 3.0+
- Camera插件
- 横屏布局
- Material Design

## 安装和运行

1. 确保已安装Flutter SDK
2. 运行 `flutter pub get` 安装依赖
3. 连接设备或启动模拟器
4. 运行 `flutter run`

## 权限说明

- 摄像头权限：用于拍摄皮肤照片
- 存储权限：用于保存拍摄的照片

## 注意事项

- 应用强制横屏显示
- 当前版本使用模拟的年龄估算算法
- 实际部署时需要集成真实的AI模型
