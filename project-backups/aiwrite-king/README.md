# AI写作助手

一个基于HarmonyOS的智能写作工具，集成了AI写作功能，帮助用户提高写作效率和质量。

## 功能特性

### 🤖 AI写作功能
- **智能生成**: 基于用户输入生成高质量的写作内容
- **文本优化**: 自动优化文本表达，提升语言质量
- **智能续写**: 根据已有内容智能续写，保持逻辑连贯
- **文本改写**: 提供多种风格的文本改写选项
- **真实AI**: 集成硅流大模型API，提供真实的AI写作体验
- **智能限流**: 每日10次AI调用限制，合理控制使用成本

### 📝 写作模板
- **创意写作**: 自由创作，发挥想象力
- **商务邮件**: 专业的商务沟通模板
- **学术论文**: 严谨的学术写作格式
- **故事创作**: 引人入胜的故事结构
- **诗歌创作**: 优美的诗歌创作模板
- **技术文档**: 清晰的技术说明格式

### 📚 文档管理
- **文档保存**: 自动保存写作内容
- **文档列表**: 管理所有写作文档
- **文档加载**: 快速打开历史文档
- **字数统计**: 实时显示文档字数
- **导出功能**: 支持多种格式导出

### ⚙️ 个性化设置
- **字体大小**: 可调节的字体大小
- **主题切换**: 支持浅色/深色主题
- **自动保存**: 可配置的自动保存功能
- **AI模型**: 多种AI模型选择
- **语言设置**: 多语言支持

## 技术架构

### 核心技术
- **HarmonyOS**: 基于HarmonyOS应用框架
- **ArkTS**: 使用ArkTS语言开发
- **ArkUI**: 现代化的UI框架
- **数据存储**: 使用preferences进行本地数据存储
- **AI集成**: 集成硅流大模型API (Qwen/QwQ-32B)
- **限流机制**: 智能的用户调用次数管理

### 项目结构
```
entry/src/main/ets/
├── pages/
│   ├── Index.ets          # 主页面
│   └── Settings.ets       # 设置页面
├── services/
│   ├── AIWritingService.ets    # AI写作服务
│   └── DocumentService.ets     # 文档管理服务
└── entryability/
    └── EntryAbility.ets   # 应用入口
```

### 服务层设计
- **AIWritingService**: 负责AI写作相关的所有功能
- **DocumentService**: 负责文档的增删改查和导出
- **模块化设计**: 便于维护和扩展

## 使用指南

### 基本使用
1. **开始写作**: 在文本编辑区域输入内容
2. **选择模板**: 从下拉菜单选择适合的写作模板
3. **AI助手**: 点击"AI助手"按钮获取智能建议
4. **保存文档**: 点击"保存文档"按钮保存内容

### AI功能使用
1. **生成建议**: 输入关键词后点击"生成建议"
2. **文本优化**: 选择文本后点击"优化"按钮
3. **智能续写**: 点击"续写"按钮继续写作
4. **文本改写**: 点击"改写"按钮改变文本风格

### 文档管理
1. **查看文档**: 点击"文档"按钮查看所有文档
2. **加载文档**: 点击文档列表中的项目加载文档
3. **新建文档**: 点击"新建文档"开始新的写作
4. **导出文档**: 点击"导出"按钮导出文档

## 开发说明

### 环境要求
- HarmonyOS SDK 4.0+
- DevEco Studio 4.0+
- Node.js 16+

### 构建步骤
1. 克隆项目到本地
2. 使用DevEco Studio打开项目
3. 配置HarmonyOS SDK
4. 运行项目到模拟器或真机

### 扩展开发
- **添加新的AI功能**: 在`AIWritingService.ets`中添加新方法
- **自定义模板**: 修改`getWritingTemplates()`方法
- **UI定制**: 修改`Index.ets`中的UI组件
- **数据存储**: 在`DocumentService.ets`中扩展存储功能
- **API配置**: 在`ApiConfig.ets`中修改API密钥和模型
- **限流设置**: 在`RateLimitService.ets`中调整限流策略
