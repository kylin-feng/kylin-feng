🚀 SmartMeet AI 手动启动指南

由于bash环境问题，请手动启动服务

📊 服务状态检查

后端服务 (http://localhost:5001)
前端服务 (http://localhost:3000)

🔧 步骤1: 启动后端服务

打开新的终端窗口，运行以下命令：

cd /Users/shixianping/smart-meet/backend
npm install
npm run dev

成功后会看到 "SmartMeet AI API 启动成功!" 消息

🎨 步骤2: 启动前端服务

打开另一个终端窗口，运行以下命令：

cd /Users/shixianping/smart-meet/frontend
npm install
npm run dev

成功后会看到 "Local: http://localhost:3000/" 消息
